NEAREST NEIGHBORS

The following libraries need to be installed : 
1) numpy
2) matplotlib
3) scipy

RUNNING INSTRUCTIONS : 

1) Place all the files in a desired location

2) In the python shell, excecute the .py files in the following order
	
	>>python mnist.py
	>>python mnist1NNdemo.py
	>>python iris1NNboundary.py

3) Output for 4-a will be appear upon running mnist1NNdemo.py
   Output for 4-b will appear as a window for the Decision boundaries, and the error rates will be displayed in the terminal. 
   Maximize the window for better visualization
